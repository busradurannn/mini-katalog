# Mini Katalog Uygulaması

Flutter ile geliştirilmiş basit ve profesyonel bir ürün katalog uygulaması.

## Özellikler
- Ürün listeleme (GridView)
- Ürün detay sayfası
- Arama ve filtreleme
- Sepete ekleme (simülasyon)
- Gerçek API entegrasyonu

## Kullanılan Teknolojiler
- Flutter 3.41.9
- Dart SDK
- Material Design 3
- Ekstra paket kullanılmamıştır

## Çalıştırma Adımları

1. Flutter SDK kurulu olmalıdır
2. Projeyi klonlayın:
   git clone <repo-url>
3. Bağımlılıkları yükleyin:
   flutter pub get
4. Uygulamayı çalıştırın:
   flutter run
