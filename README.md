# 🛍️ Mini Katalog Uygulaması

Flutter ile geliştirilmiş, temel mobil uygulama geliştirme becerilerini sergilemek amacıyla tasarlanmış bir ürün katalog uygulaması.

---

## 📱 Uygulama Hakkında

Bu uygulama; ürün listeleme, detay sayfası ve basit sepet sistemi özelliklerini barındıran bir mini e-ticaret kataloğudur. Flutter'ın temel widget yapısı, navigasyon sistemi ve veri modelleme özellikleri kullanılarak geliştirilmiştir.

### Özellikler
- Ana sayfa ile karşılama ekranı
- GridView tabanlı ürün listeleme
- Ürün detay sayfası
- Sayfalar arası Navigator ile geçiş
- Route Arguments ile veri aktarımı
- Basit sepet sistemi simülasyonu
- JSON tabanlı veri modeli (fromJson / toJson)
- Dinamik liste oluşturma (ListView.builder)

## 🚀 Çalıştırma Adımları

### Gereksinimler
- Flutter SDK kurulu olmalıdır → [Flutter Kurulum Rehberi](https://docs.flutter.dev/get-started/install)
- Android Studio veya VS Code yüklü olmalıdır
- Android Emulator veya fiziksel bir Android cihaz bağlı olmalıdır

### Kurulum

1. **Repoyu klonlayın:**
   ```bash
   git clone https://github.com/KULLANICI_ADINIZ/REPO_ADINIZ.git
   cd REPO_ADINIZ
   ```

2. **Bağımlılıkları yükleyin:**
   ```bash
   flutter pub get
   ```

3. **Emülatörü başlatın** (Android Studio üzerinden) veya fiziksel cihazı bağlayın.

4. **Uygulamayı çalıştırın:**
   ```bash
   flutter run
   ```
